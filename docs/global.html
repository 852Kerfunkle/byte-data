fromBytes(buffer, bitDepth, options) → {!Array.<number>|string}

Turn a byte buffer into what the bytes represent.
Parameters:
Name Type Description
buffer !Array.<number> | Uint8Array An array of bytes.
bitDepth number The bit depth of the data. Possible values are 1, 2, 4, 8, 16, 24, 32, 40, 48 or 64.
options Object The options. They are: - "signed": If the numbers are signed. Default is false (unsigned). - "float": True for floating point numbers. Default is false. This option is available for 16, 32 and 64-bit numbers. - "base": The base of the input. Default is 10. Can be 2, 10 or 16. - "char": If the bytes represent a string. Default is false. - "be": If the values are big endian. Default is false (little endian). - "single": If it should return a single value instead of an array. Default is false.
